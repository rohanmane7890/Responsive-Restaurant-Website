## RoyalX Luxury Hotel - React Website

# Project Overview
A modern, responsive luxury hotel website built with React 19 and Vite. Features elegant design with interactive components, smooth animations, and mobile responsiveness.

# Features

  Design & UI

 Modern, elegant luxury hotel theme

 Fully responsive design (Mobile, Tablet, Desktop)

 Smooth animations and transitions

 Clean, intuitive navigation

 Color palette: Gold (#9d7e54) & Dark themes

#  Technical Features

 Built with React 19 + Vite

 CSS Modules for scoped styling

 Swiper.js for carousels

 React Hooks for state management

 Remixicon integration

 ESLint configuration

#  Interactive Components
   Responsive navigation with scroll effects

   Image carousels with autoplay

   3D flip cards for room displays

   Testimonial sliders

   Amenities showcase

   Services grid with hover effects

# Frontend

React 19 - UI Library

Vite - Build tool & dev server

CSS Modules - Scoped styling

Swiper.js - Touch slider

Remixicon - Icon library

Google Fonts - Playfair & Poppins

# Styling
